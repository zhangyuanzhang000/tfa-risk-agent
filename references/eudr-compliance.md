# EUDR（欧盟零毁林法规）合规参考

## 法规概述

**全称**：EU Deforestation Regulation (EUDR)
**法规号**：Regulation (EU) 2023/1115
**生效日期**：2023年6月29日
**强制执行日期**：
- 大型企业：2024年12月30日（延期至2025年12月30日）
- 中小企业：2025年6月30日（延期至2026年6月30日）

## 适用范围

### 覆盖商品

| 商品 | HS编码范围 |
|-----|-----------|
| 棕榈油 | 1511, 1513, 1516, 2306.50 |
| 大豆 | 1201, 1507, 2304 |
| 牛 | 0102, 0201, 0202, 0206.10 |
| 咖啡 | 0901 |
| 可可 | 1801, 1803, 1804, 1805, 1806 |
| 橡胶 | 4001, 4002, 4003, 4005 |
| 木材及木制品 | 44章 |

### 覆盖产品

- 原生商品
- 含有上述商品的加工产品
- 衍生产品（如皮革、巧克力、家具）

## 核心要求

### 1. 零毁林定义

**关键日期**：2020年12月31日

产品必须满足：
- 不包含在2020年12月31日后毁林土地上生产的商品
- 遵守生产国相关法律法规

### 2. 尽职调查义务

**三要素**：

#### 信息收集（Information）

必须收集并保存5年：
- 产品描述和数量
- 供应商信息
- 生产国/地区
- 地理定位数据（Geo-location）
- 生产日期/收割日期

#### 风险评估（Risk Assessment）

使用以下标准评估：
- 生产国毁林风险评级（欧盟将发布风险分类）
- 供应链复杂性
- 文件造假风险
- 与土著人民冲突历史
- 联合国制裁

#### 风险缓解（Risk Mitigation）

高风险情况需：
- 增原产地证据
- 独立审计
- 卫星监测
- 现场核查

### 3. 地理定位数据要求

| 农场面积 | 要求 |
|---------|-----|
| <4公顷 | 至少一个GPS点（多边形或点） |
| ≥4公顷 | 完整多边形边界 |

**数据格式**：
- 经纬度（WGS84坐标系）
- 多边形文件（GeoJSON, Shapefile等）

### 4. 合规声明（Due Diligence Statement）

**要求**：
- 在欧盟海关信息系统（ICS2）中提交
- 首次投放市场或出口前提交
- 包含尽职调查结论
- 关联参考编号

## 风险分级体系

欧盟将国家分为三级：

| 级别 | 定义 | 尽职调查要求 |
|-----|-----|-------------|
| **低风险** | 毁林风险低，治理良好 | 简化程序 |
| **标准风险** | 中等风险 | 标准尽职调查 |
| **高风险** | 毁林风险高，治理薄弱 | 强化尽职调查 |

**高风险国家示例**（预期）：
- 巴西（塞拉多、亚马逊）
- 印尼（苏门答腊、加里曼丹）
- 刚果民主共和国
- 喀麦隆

## 合规检查清单

### 企业自查清单

- [ ] 建立EUDR合规体系
- [ ] 识别适用的产品和HS编码
- [ ] 建立供应商信息收集系统
- [ ] 获取农场/生产地地理定位数据
- [ ] 建立风险评估程序
- [ ] 制定风险缓解措施
- [ ] 培训相关人员
- [ ] 建立文件存档系统（5年）
- [ ] 注册欧盟海关系统（EORI）

### 供应商评估清单

- [ ] 供应商是否了解EUDR要求？
- [ ] 能否提供地理定位数据？
- [ ] 能否证明2020年后无毁林？
- [ ] 是否遵守生产国法律？
- [ ] 是否有独立认证（如RSPO、RTRS）？

## 处罚措施

**违规后果**：
- 产品没收
- 产品销毁
- 罚款（最高可达欧盟年营业额4%）
- 排除在公共采购之外
- 禁止市场准入

## 与其他标准的关系

| 标准 | 与EUDR关系 |
|-----|-----------|
| **TFA DDQs** | EUDR尽职调查可借鉴TFA框架 |
| **RSPO/RTRS** | 认证可作为合规证据，但不等于自动合规 |
| **FSC** | 木材认证可作为EUDR合规支持 |
| **CDP Forests** | 披露内容可支持EUDR准备 |

```
【EUDR合规改进建议】

现状：[描述当前状况]

EUDR要求：[引用具体条款]

差距分析：
- [差距1]
- [差距2]

建议措施：
1. [措施1]
2. [措施2]

优先级：[高/中/低]
时间要求：[EUDR强制执行日期]
```
