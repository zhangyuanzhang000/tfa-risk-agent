# 大豆专属评估标准

## 供应链结构

### 直接采购 vs 间接采购

```
┌────────────────────────────────────────────────────────────┐
│                    大豆供应链层级                          │
├────────────────────────────────────────────────────────────┤
│  Tier 1: 农场/种植者                                         │
│     ↓                                                      │
│  Tier 2: 收购商/本地贸易商                                   │
│     ↓                                                      │
│  Tier 3: 出口商/大型贸易商（四大粮商）                        │
│     ↓                                                      │
│  Tier 4: 压榨厂/加工企业                                     │
│     ↓                                                      │
│  Tier 5: 终端用户（食品/饲料企业）                           │
└────────────────────────────────────────────────────────────┘
```

**直接采购**：企业与Tier 3+直接签约
**间接采购**：通过Tier 4/5采购，无法直接联系上游

### 风险阈值

| 直接采购比例 | 风险等级 | 评估建议 |
|-------------|---------|---------|
| >70% | 🟢 低风险 | 溯源可行，可要求供应商合规 |
| 30-70% | 🟡 中风险 | 需加强贸易商管理 |
| <30% | 🔴 高风险 | 溯源困难，依赖贸易商自我声明 |

## 关键风险地区

### 巴西（全球第一大大豆出口国）

#### 亚马逊生物群系
- **风险等级**：🔴 极高
- **核心问题**：2008年后新毁林土地上种植的大豆
- **法规要求**：亚马逊大豆禁运协议（Amazon Soy Moratorium）
- **验证要点**：
  - 供应商是否在禁运区有新开发农场
  - 是否有2008年后毁林的土地证书

#### 塞拉多生物群系
- **风险等级**：🔴 极高
- **核心问题**：毁林速度已超亚马逊，但无同等禁运保护
- **关注指标**：
  - 是否有塞拉多专项保护承诺
  - 是否参与塞拉多保护联盟（Cerrado Manifesto）

#### 潘塔纳尔生物群系
- **风险等级**：🟡 中高
- **核心问题**：湿地生态系统脆弱，火灾风险

#### 大西洋森林
- **风险等级**：🔴 极高
- **核心问题**：极度濒危生物群系，残余面积<10%

### 美国（全球第二大大豆出口国）

- **风险等级**：🟡 中等
- **主要风险**：
  - 草原转换（Prairie Pothole Region）
  - 水资源使用
  - 土壤退化

### 阿根廷（全球第三大大豆出口国）

- **风险等级**：🟡 中高
- **主要风险**：
  - 查科森林（Gran Chaco）毁林
  - 湿地排水

### 中国（主要生产国）

- **风险等级**：🟡 中等
- **主要地区**：黑龙江、吉林、辽宁、内蒙古
- **关键问题**：
  - 非转基因大豆溯源
  - 黑土地保护

### 其他生产国

| 国家 | 风险等级 | 关键问题 |
|-----|---------|---------|
| 巴拉圭 | 🔴 高 | 查科森林毁林 |
| 乌拉圭 | 🟢 低 | 主要是草原转换 |
| 加拿大 | 🟢 低 | 主要是泥炭地排水 |
| 乌克兰 | 🟡 中 | 草原转换 |

## 关键法规与标准

### 巴西森林法（Forest Code）

**核心要求**：
- 亚马逊：Legal Reserve（LR）80%，APP（永久保护区）需保留
- 塞拉多：LR 35%
- 其他：LR 20%

**验证文件**：
- CAR（Cadastro Ambiental Rural）- 农村环境登记
- CCU（Cadastro de Imóveis Certificados）- 认证财产登记

### 亚马逊大豆禁运协议（Amazon Soy Moratorium）

**覆盖范围**：巴西亚马逊生物群系
**核心承诺**：
- 不采购2008年7月后在亚马逊新毁林土地上种植的大豆
- 每年第三方审计

**验证方式**：
- 供应商是否在禁运合规名单中
- 卫星图像验证

### 塞拉多保护联盟（Cerrado Manifesto）

**性质**：自愿承诺（非强制性禁运）
**签署企业**：260+国际企业
**核心要求**：
- 呼吁政府加强塞拉多保护
- 供应链中避免塞拉多毁林

## 大豆认证体系

| 认证 | 覆盖内容 | 市场认可度 | 验证难度 |
|-----|---------|-----------|---------|
| **RTRS** | 大豆专属，覆盖环境+社会 | 高 | 中 |
| **ProTerra** | 非转基因+可持续 | 中 | 中 |
| **ISCC** | 生物燃料可持续认证 | 中 | 低 |
| **2BSvs** | 欧盟生物燃料认证 | 中 | 低 |
| **非转基因** | 仅转基因状态 | 高 | 低 |

### RTRS（负责任大豆圆桌会议）

**标准覆盖**：
- 合法合规
- 负责任劳工实践
- 社区关系
- 环境责任
- 良好农业实践

**认证级别**：
- 大规模农场认证
- 小农户团体认证
- 供应链证书（CoC）

## 评估清单

### 高风险信号（立即标记）

- [ ] 巴西亚马逊/塞拉多采购，无NDPE承诺
- [ ] 无法识别主要供应商
- [ ] 无巴西森林法合规证明要求
- [ ] 间接采购比例>70%
- [ ] 无大豆禁运合规验证

### 中风险信号（需关注）

- [ ] 有NDPE承诺但无具体实施数据
- [ ] 仅能追溯至国家层面
- [ ] 依赖贸易商声明，无独立验证
- [ ] 无小农户支持计划

### 低风险特征（良好实践）

- [ ] 公开NDPE政策，设定明确时间表
- [ ] 供应链透明，公布主要供应商名单
- [ ] RTRS或其他认证覆盖>50%
- [ ] 农场级溯源系统运行中
- [ ] 定期发布可持续采购进展报告
