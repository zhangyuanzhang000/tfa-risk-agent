# CSRD（企业可持续发展报告指令）参考

## 法规概述

**全称**：Corporate Sustainability Reporting Directive (CSRD)
**法规号**：Directive (EU) 2022/2464
**生效日期**：2023年1月5日
**适用范围**：欧盟及在欧盟有业务的非欧盟企业

## 实施时间表

| 报告年份 | 适用对象 |
|---------|---------|
| 2024 | 已受NFRD约束的大型公共利益实体（约11,700家） |
| 2025 | 所有大型欧盟企业（>250员工，或>€4000万营收，或>€2000万资产） |
| 2026 | 上市的中小企业（除微型企业） |
| 2028 | 在欧盟有业务且年营收>€1.5亿的非欧盟企业 |

## 核心要求

### 双重重要性（Double Materiality）

**财务重要性**：可持续发展问题对企业财务状况的影响
**影响重要性**：企业活动对环境和社会的影响

### ESRS标准体系

**欧洲可持续发展报告标准（European Sustainability Reporting Standards）**

#### 通用标准（Cross-cutting）

| 标准 | 内容 |
|-----|-----|
| ESRS 1 | 通用要求 |
| ESRS 2 | 一般披露（强制） |

#### 环境标准（E）

| 标准 | 主题 | 与森林风险商品相关 |
|-----|-----|------------------|
| ESRS E1 | 气候变化 | ⭐ 土地利用变化排放 |
| ESRS E2 | 污染 | - |
| ESRS E3 | 水资源 | - |
| ESRS E4 | 生物多样性和生态系统 | ⭐⭐ 毁林直接影响 |
| ESRS E5 | 资源利用和循环经济 | ⭐ 可持续采购 |

#### 社会标准（S）

| 标准 | 主题 | 与森林风险商品相关 |
|-----|-----|------------------|
| ESRS S1 | 自有员工 | - |
| ESRS S2 | 价值链中的工人 | ⭐⭐ 小农户、农场劳工 |
| ESRS S3 | 受影响社区 | ⭐⭐ 原住民、当地社区 |
| ESRS S4 | 消费者和终端用户 | - |

#### 治理标准（G）

| 标准 | 主题 |
|-----|-----|
| ESRS G1 | 商业行为 |

## 与TFA评估的关联

### 生物多样性和生态系统（ESRS E4）

**披露要求**：
- 生物多样性足迹
- 土地利用变化影响
- 生态系统服务依赖

**TFA评估数据支持**：
- 采购来源地区生物多样性价值
- 毁林面积估算
- 高保护价值区域影响

### 气候变化（ESRS E1）

**披露要求**：
- 范围3排放（供应链）
- 土地利用变化排放

**TFA评估数据支持**：
- 供应链碳足迹
- 毁林相关排放
- 可持续采购减排贡献

### 价值链中的工人（ESRS S2）

**披露要求**：
- 强迫劳动风险
- 童工风险
- 工作条件

**TFA评估数据支持**：
- 农场劳工权益状况
- 小农户生计影响
- 认证体系劳工标准

### 受影响社区（ESRS S3）

**披露要求**：
- 土地权利
- 原住民权利
- 社区影响

**TFA评估数据支持**：
- 原住民土地冲突
- 社区申诉机制
- 利益相关方参与

## 价值链视角要求

CSRD要求报告**整个价值链**的影响，包括：
- 上游：原材料采购（森林风险商品）
- 下游：产品使用和处置

这对依赖软性商品的企业尤为重要。

## 报告格式

- **数字报告**：必须以XHTML格式提交
- **标记语言**：必须使用ESEF（欧洲单一电子格式）或内联XHTML
- **数据可机读**：结构化数据便于比较

## 审计要求

- **有限保证**：初期要求
- **合理保证**：逐步过渡（预计2028年后）

## 评估输出模板

当在TFA评估中发现CSRD相关问题时：

```
【CSRD披露准备建议】

相关ESRS标准：[如ESRS E4, ESRS S2等]

当前披露状态：[描述]

数据缺口：
- [缺口1]
- [缺口2]

建议收集的数据：
1. [数据需求1]
2. [数据需求2]

报告整合建议：
- 如何将TFA评估结果整合到CSRD报告中

时间要求：[适用报告年份]
```
